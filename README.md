## 🚀 Free Online Image Converter

PixConvert is a modern **online image converter** that allows you to convert images between popular formats directly inside your browser.

Unlike many image conversion tools, PixConvert performs all processing locally. Your files never leave your device, ensuring maximum privacy and fast conversion speeds.

### Key features

- ⚡ Lightning-fast image conversion
- 🔒 100% local processing (no uploads)
- 🖼 Support for modern and legacy image formats
- 📱 Responsive design for desktop and mobile
- 🎯 Batch-friendly workflow
- 🚫 No registration required
- 💻 Open source
- 🌐 Cross-platform support
- 🔥 Built with React, Vite, and TypeScript

# 📦 Local Development

Clone the repository:

```bash
git clone https://github.com/Edo911/pix-convert.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

---

# 🚀 Production Build

Build the application:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

Deploy the generated `dist/` directory to:

- Vercel
- Netlify
- Cloudflare Pages
- GitHub Pages
- Firebase Hosting
